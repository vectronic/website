## Environment Setup

Clone the repository locally.

Working from the root of the project, install required software (assuming MacOS):

```
brew install hugo
npm install netlify-cli -g
netlify link
```

## Development

`npm run build`

or:

`netlify build`

or:

`npm run watch`
